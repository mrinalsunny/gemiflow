# hive-mind-init

Initialize the Hive Mind collective intelligence system.

## Usage
```bash
npx gemiflow4gemini hive-mind init [options]
```

## Options
- `--force` - Force reinitialize
- `--config <file>` - Configuration file

## Examples
```bash
npx gemiflow4gemini hive-mind init
npx gemiflow4gemini hive-mind init --force
```
